# Magyar

Minecraft server website for server members.

## Technologies

- [Next.js]
- [NextAuth.js]
- [Prisma]
- [Tailwind CSS]
- [tRPC]
- [Typescript]
- [React]

## Features

- multi-step form.
- admin can view forms and create users.
- you can see your skin on your dashboard.
